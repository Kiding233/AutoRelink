# AutoRelink

Project Graph 扩展：拖拽节点时自动调整连线端点位置。支持 1000+ 节点大图，八方向算法实时判断连接方向。

## 安装

下载 [Release](https://github.com/Kiding233/AutoRelink/releases) 中的 `.prg` 文件，拖入 Project Graph 窗口即可。

## 快捷键

- `a r l` — 开关插件
- `a r d` — 显示调试日志
- `a r p` — 显示性能分析

## 开发

```bash
pnpm install      # 安装依赖
pnpm dev          # 开发模式（热重载）
pnpm build        # 构建
pnpm package      # 打包
pnpm install:ext  # 安装到本地 Project Graph
```
